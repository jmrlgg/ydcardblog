# ydcardblog

Blog with Card Displays and Sidebar
===================================

Features
--------

1. NavWalker Menu
2. Email Newsletter Built In
   * After Single Post
    * In Sidebar 
    * Possible Built into Footer 
3. Custom Header with Label for Each page
4. TAGS Shown in Header
